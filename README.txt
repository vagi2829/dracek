# STAVBA+ — moderní one‑page template (HTML/CSS/JS)

## Spuštění
1) Rozbal ZIP
2) Otevři `index.html` v prohlížeči

(Template nepoužívá build systém ani knihovny.)

## Co tu je
- Sticky topbar + mobilní drawer
- Reveal animace na scroll (IntersectionObserver)
- Tilt/parallax hero karta
- Animované počítadla
- Filtr nabídky (demo data)
- Modaly (služby, projekty, nabídky)
- Carousel reference + progress bar
- Toast notifikace
- Šedá + červená estetika

## Úpravy
- Data nabídek v `app.js` (pole `listings`)
- Texty v `index.html`
- Barvy v `styles.css` (`:root`)

Pozn.: Fotky jsou z Unsplash (přímé URL).
